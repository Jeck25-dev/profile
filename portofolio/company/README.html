Catatan Penting

Semua data bersifat statis. Tidak ada database, backend, atau server yang dibutuhkan. Semua file bisa langsung dibuka di browser cukup dengan double-click file HTML.
Sesi login menggunakan sessionStorage. Data sesi tersimpan sementara di browser. Setelah tab atau browser ditutup, sesi otomatis terhapus.
Navigasi sidebar sudah saling terhubung. Link antar halaman sudah aktif: Dasbor Admin ↔ Proyek ↔ Pengguna ↔ Tagihan.
Font membutuhkan koneksi internet. Google Fonts (Cormorant Garamond + Montserrat) di-load dari CDN. Untuk demo offline, unduh font terlebih dahulu.
Semua tombol aksi sudah terpasang. Tombol seperti "Simpan", "Hapus", "Ekspor" menggunakan alert() sebagai placeholder — mudah diganti fungsi nyata.
Mudah dikustomisasi. Nama perusahaan "Nusantara Group" bisa diganti di semua file. Warna utama diatur lewat variabel CSS --gold: #C9A84C di bagian :root di setiap file.
Versi Bahasa Indonesia. Seluruh konten termasuk label, navigasi, placeholder form, pesan sistem, dan data demo sudah diterjemahkan ke Bahasa Indonesia. Mata uang menggunakan Rupiah (Rp), lokasi Jakarta, dan kontak disesuaikan.
